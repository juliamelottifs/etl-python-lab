Projeto ETL com Python

Projeto desenvolvido para praticar o processo de ETL (Extração, Transformação e Carregamento) utilizando Python e Pandas.

Os dados dos usuários foram criados localmente em uma lista Python, transformados com mensagens personalizadas e exportados para um arquivo CSV, simulando um fluxo real de tratamento de dados.

Tecnologias utilizadas

🐍 Python
🐼 Pandas
💻 VS Code
🐙 GitHub
📍 Etapas do projeto
📥 Extração

Criação de dados fictícios de usuários.

Transformação

Geração de mensagens personalizadas automaticamente.

Carregamento

Exportação dos dados tratados para um arquivo CSV.
